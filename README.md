# Crestal Zone Mockup

A modern, Material Design 3 (MD3) inspired mockup for a smart carpark management system.

## 🚀 Overview

Crestal Zone is a comprehensive solution for managing residential carpark access, featuring three distinct portals:

1.  **Operator Dashboard**: Real-time monitoring of gate entries/exits, license plate recognition (LPR) confidence tracking, and manual override controls.
2.  **Resident Portal**: Self-service management for residents to register their vehicles and invite guests via WhatsApp-linked invitations.
3.  **Guest Self-Service**: A mobile-optimized view for guests to provide their vehicle plate number before or after entering the premises.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS + Inline Styles (MD3 Theming)
- **Design System**: Material Design 3

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd zone-mockup
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
